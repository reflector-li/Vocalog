# Changelog

All notable changes to Vocalog will be documented in this file.

## [1.0.0] - 2025-03-06

### Added
- STT API integration for audio transcription (OpenAI Whisper compatible)
- LLM-powered note summarization (OpenAI Chat compatible)
- Automatic daily note writing
- Calendar interface with multi-date selection
- Quick select buttons (Today, Yesterday, This Week, Last 7 Days)
- Month navigation in calendar view
- Date range selection for batch processing
- Selected file processing from file explorer
- Context menu integration for audio files
- Enhanced error handling with detailed messages
- Ribbon icon and command palette integration
